# searchT (ThreadsCollector)

> Meta Threads 빅데이터 수집 & 분석 플랫폼

## Features
- Hybrid Collection Engine: Meta Official API + Web Scraper
- AI Category Classifier: Shopping / Issue / Personal  
- Affiliate Link Detector: AliExpress, Coupang, Rakuten, Amazon
- Sentiment Analysis & Trending Keywords
- Real-time Dashboard: Express REST API

## Quick Start
docker-compose up -d
npm install && cp .env.example .env
npm run seed && npm run dev

## Tech Stack
Node.js, Express, MongoDB, Redis, BullMQ, OpenAI, Docker, Railway
